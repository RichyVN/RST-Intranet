Omits Parent Page Templates from the "Page" edit screen in the WordPress admin.

== Description ==

This plugin does what it names says; it omits page templates from the parent theme from the template dropdown when you are editing a "Page" in the WordPress admin.

== Frequently Asked Questions ==

= What do I do after activation? =

Nothing besides use WordPress as you normally would. The plugin simply filters out the page templates of the parent theme from the template dropdown when you are editing a "Page" in the WordPress admin.

= Where is the Settings Page? =

There is no settings pages. To enable this plugin, activate in _Plugins > Installed Plugins_ in the WordPress Admin. To disable, deactivate the plugin.
